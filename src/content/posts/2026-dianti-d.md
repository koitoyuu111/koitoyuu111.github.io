## 项目概述

云台端搭载 K230 视觉模块，通过 cv_lite 矩形检测识别 **2026 电赛 E 题靶子**，计算矩形中心点坐标偏移，通过串口将靶子位置实时发送给小车。小车端采用 STM32F407 主控，基于速度环 PID 实现精确运动控制，并加入圆弧路径规划算法使行驶轨迹更平滑。两机协同完成靶子跟踪与动态接近任务。

项目包含两个子工程：
- **云台**：矩形靶子识别 + DM4310 伺服电机控制
- **小车**：运动控制 + 圆弧路径规划

## K230 视觉识别 (矩形8.py)

识别 2026 电赛 E 题矩形靶子，通过 UART2 (921600bps) 与 STM32 通信：

### 矩形检测流程

```
传感器捕获 (480×320 RGB888)
  → Canny 边缘检测 (低阈值 40, 高阈值 150)
  → cv_lite 矩形检测
  → 高斯模糊 (kernel=3)
  → 多边形拟合 (epsilon=0.02)
  → 最大矩形筛选 (面积阈值 2500)
  → 对边平行/垂直验证 (容差 30°)
  → 对角线交点计算 = 矩形中心点
  → 中心偏移 (x-240, 160-y) 发送
```

### 检测参数

| 参数 | 值 | 说明 |
|------|-----|------|
| canny_thresh1 | 40 | Canny 低阈值 |
| canny_thresh2 | 150 | Canny 高阈值 |
| approx_epsilon | 0.02 | 多边形拟合精度 |
| length_threshold | 120 | 对边差值容限 (像素) |
| S_THRESHOLD | 2500 | 矩形面积阈值 |

### 通信协议 (K230 → STM32)

| 指令 | 含义 |
|------|------|
| `0xFF 0xFE 0x01 [X偏移 int16]` | 靶子中心 X 偏移 (图像中心 240) |
| `0xFF 0xFE 0x02 [Y偏移 int16]` | 靶子中心 Y 偏移 (图像中心 160) |
| `0xFF 0xFE 0x03 0x0001` | 未检测到矩形靶子 |

## 子工程一：云台

### 外设清单

| 外设 | 实例 | 功能 |
|------|------|------|
| CAN | CAN1 | DM4310 伺服电机通信 |
| USART | USART1 | 与小车通信 |
| USART | USART2 | K230 视觉模块 (921600bps) |
| USART | USART3 | WT9011 姿态传感器 (921600bps) |
| USART | USART6 | VOFA 调试 (38400bps) |
| TIM | TIM8 | 主控制循环 (1000Hz) |

### DM4310 伺服电机 (CAN 总线)

通过 CAN1 总线控制两个 DM4310 电机：
- `ctrl_enable(id)` / `ctrl_disable(id)` — 使能/关闭
- `speed_ctrl(hcan, id, speed)` — 速度控制
- 状态反馈：`motor[id].para.pos / vel / state`

### PID 控制系统

| PID 实例 | 用途 |
|----------|------|
| Y_PID | Y 方向 (前后) 速度环 |
| X_PID | X 方向 (左右) 位置环 |
| Y_POS_PID | Y 方向位置环 (升降控制) |
| X_POS_PID | X 方向位置环 |
| Angl | 角度环 |
| W_Angl | 角速度环 (WT9011 陀螺仪反馈) |

### 任务 (TIM8 中断 @ 1000Hz)

| 任务 | 功能 |
|------|------|
| task_2 | 位置环升降 + 靶子识别 + 低速 PID 接近 |
| task_3 | 位置环升降 + 靶子识别 + PID 接近 (不同参数) |
| task_4 | 动态 X 目标递减 + PID 大参数接近靶子 |
| task_6 | 带陀螺仪角速度反馈的动态靶子追踪 |

## 子工程二：小车

### 控制架构

- 接收云台端 K230 矩形靶子识别数据 (蓝牙/串口)
- 速度环 PID 精确运动控制
- 圆弧路径规划 → 平滑接近靶子
- 两机协同靶子追踪

## 系统通信链路

```
K230 视觉(矩形8.py) ──(USART2 921600)──→ 云台 STM32F407
                                            │
                                  CAN1 ──→ DM4310 电机 x2
                                            │
                                  USART1 ──→ 小车 STM32F407
                                            │
               ──(CAN1)──→ 电机    ──(WT9011)──→ IMU
               ──(蓝牙)──→ 蓝牙手柄
```

## 核心算法

### PID 多环控制

| 控制环 | 传感器 | 输出 |
|--------|--------|------|
| 速度环 | 电机编码器 | 速度 PWM |
| 位置环 | K230 矩形靶子坐标 | 速度指令 |
| 角度环 | WT9011 Yaw | 差速修正 |
| 角速度环 | WT9011 Wz | 角度阻尼 |

### 圆弧路径规划

小车端实现圆弧路径规划算法，将直线目标分解为平滑弧线轨迹，避免直角转弯的惯性冲击，使接近靶子的运动更平滑流畅。
