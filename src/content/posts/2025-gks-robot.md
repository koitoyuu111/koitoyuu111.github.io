## 项目概述

基于 STM32F407 + ZDT_X42_V2 闭环步进伺服的排爆机器人系统。搭载 K230 视觉模块，通过 AI 模型识别 **人质靶子**（圆柱形/圆锥形/腰鼓形），并结合颜色追踪、二维码解析、圆形排爆桶检测，WT9011 IMU 姿态反馈与 35+ 状态机实现全自动任务流程。麦轮运动学解算 + PID 角度/位置双环控制，蓝牙与机械臂子系统协同完成自主抓取与放置。视觉矫正与无视觉矫正双模式适配不同场景。

项目包含两个版本：
- **GKS_max - 视觉**：完整视觉识别版本
- **GKS_max - 无视觉矫正**：去掉视觉矫正的简化版本

## K230 视觉识别

### demo1_1.py — AI 人质靶子识别

通过 nncase 部署 AI 模型（kmodel），对三种人质靶子进行识别：

| 形状编号 | 人质靶子类型 | 置信度阈值 |
|----------|-------------|-----------|
| 1 | 圆柱形人质 | ≥0.5 |
| 2 | 圆锥形人质 | ≥0.5 |
| 3 | 腰鼓形人质 | ≥0.7 |

同时支持：
- **二维码解析**：扫描二维码获取 `[形状编号, 颜色编号, 模式]` 三参数
- **颜色色块追踪**：红/绿/蓝 三色 `find_blobs` (area_threshold=6000)
- **激光控制**：GPIO47 控制激光指示器
- 通过 UART3 (9600bps) 与 STM32 通信

### yssb_3.py — 颜色 + 排爆桶 + 巡线

| 功能 | 方法 | 说明 |
|------|------|------|
| 颜色追踪 | `find_blobs` | 红/绿/蓝/白/紫 5色，area_threshold=1000 |
| 圆形排爆桶 | `find_circles(threshold=1800)` | 霍夫圆检测，半径 40-70px |
| 巡线 | `find_line_segments` | 垂直方向线段 (80°-100°)，长度 ≥250px |

通信协议：

| 方向 | 指令 | 含义 |
|------|------|------|
| STM32→K230 | 0x01 + 颜色编号 | 选择目标颜色 (1=红,2=绿,3=蓝,4=白,5=紫) |
| STM32→K230 | 0x08 | 开启圆形排爆桶检测 |
| K230→STM32 | 0x013 + X偏移 (int16) | 颜色靶子 X 偏移 |
| K230→STM32 | 0x11 + X偏移 | 圆形桶 X 偏移 |

## ZDT_X42_V2 伺服驱动器

4轮独立驱动的闭环步进电机系统，通过 UART5 发送指令协议：

| 控制模式 | 功能 |
|----------|------|
| 位置控制 | `ZDT_X42_V2_Bypass_Position_LV_Control` |
| 速度控制 | `ZDT_X42_V2_Velocity_Control` |
| 力矩控制 | `ZDT_X42_V2_Torque_Control` |
| 同步运动 | `ZDT_X42_V2_Synchronous_motion` |

## 任务系统 (35+ 状态机)

任务在 main() 主循环中通过 `task_flag` 状态机切换：

| 阶段 | 状态 | 描述 |
|------|------|------|
| 启动区 | case 0-2 | 移动到二维码区域，K230 识别二维码获取任务参数 |
| 转向区 | case 2-44 | 角度环调整，原地旋转对准目标角度 |
| 爬坡区 | case 4→134→144→145 | 斜坡控制，巡线检测，路口识别 |
| 人质区 | case 5-10 | 角度环前进 + PID 对准，AI 人质靶子识别 |
| 排爆桶区 | case 11-15 | 二维码定位 (X/Y)，圆周运动，圆形桶检测 |
| 颜色/人质区 | case 16-27 | 角度对准，人质 + 颜色靶子，白色停车线 |
| 结束区 | case 27-28 | 颜色停止区检测，最终停车 |

## 麦轮运动学

```
Target_L2 = -Vx + Vy - Vz × (a + b)
Target_L1 = +Vx + Vy - Vz × (a + b)
Target_R1 = -Vx + Vy + Vz × (a + b)
Target_R2 = +Vx + Vz + Vz × (a + b)
其中 a = 0.095, b = 0.086 (轮距参数)
```

## 机械臂蓝牙通信

通过 USART1 (115200bps) 蓝牙与机械臂子系统通信：
- 接收：0x01=抓取完成, 0x02=放置完成, 0x03=人质识别完成
- 发送：各阶段到位信号 (0x01-0x16)

## 两个版本差异

| 特性 | 视觉版 | 无视觉矫正版 |
|------|--------|-------------|
| K230 视觉矫正 | 预留注释掉 | 启用 |
